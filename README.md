# Census Income Classification & Segmentation

## Requirements

```
pip install pandas numpy scikit-learn matplotlib seaborn jupyter
```

## Run

```bash
jupyter notebook census_analysis.ipynb
```

Then: **Kernel → Restart & Run All**

Estimated runtime: 5–15 minutes.

## Notebook Structure

`census_analysis.ipynb` contains all code for training and evaluating the classification model, and generating the segmentation model.

1. Data Loading & Exploration
2. Preprocessing
3. Classification Model
4. Segmentation Model
